### ✅ Q1: 出差住宿标准是多少？

**耗时**: 31.33秒

**回答**:

```
出差住宿标准分国内和国际两种情况：
一、国内出差住宿标准：
1. 一线城市（北京、上海、广州、深圳）：不超过500元/晚，重要商务活动期间经审批可适当上浮；
2. 二线城市（杭州、南京、成都、武汉、西安等省会城市）：不超过400元/晚；
3. 三线及以下城市：不超过300元/晚；
4. 特殊地区（西藏、新疆、青海等高原地区）：标准可上浮20%。

二、国际出差住宿标准：根据目的地国家和城市有不同标准，详细可查看《国际差旅管理办法》或咨询人力资源部。

### ✅ Q4: 机票应该提前多久预订？有什么注意事项？

**耗时**: 35.54秒

**回答**:

```
### 机票预订提前时间建议
1. 国内机票：建议提前7-14天预订，价格通常最优惠
2. 国际机票：建议提前21-30天预订
3. 热门航线（如北京-上海）：建议提前3-5天即可
4. 旺季出行（如春节、国庆）：建议提前1-2个月预订

### 机票预订注意事项
1. **预订渠道**：统一通过阿里商旅平台预订，平台具备企业协议价、自动审批、统一管理等优势，且提供24小时客服支持（400-800-8888）
2. **航班选择**：对比不同时段航班价格，选择性价比高的航班；避开高峰时段（如周一早上、周五晚上）可节省费用；需匹配行程安排，预留至少2小时的转机/中转时间
3. **退改签政策**：改签按照航空公司规定执行，多数特价票不可改签或改签费用较高；退票费用根据票价折扣和退票时间决定，购票时需留意退改签政策，必要时选择可退改的机票
4. **选座服务**：大部分航空公司提供免费在线选座（费用由公司报销）；如需特定座位（如紧急出口、前排）可能需付费（费用自付）；有特殊需求（如轮椅、特殊餐食）可在预订时备注
5. **值机与出行准备**：航班起飞前24小时开放在线值机，建议尽早办理；国内航班建议提前1.5小时到达机场，国际航班提前3小时到达；需要托运行李的需提前到机场办理托运手续
```

### ✅ Q9: 航班取消了怎么办？紧急情况联系谁？

**耗时**: 43.58秒

**回答**:

```
### 航班取消后的处理步骤：
1. 立即联系航空公司了解取消原因和退改方案；
2. 评估对行程的影响：
   - 若为紧急公务，考虑改签最近的航班（可能需中转或承担更高费用）；
   - 若时间允许，可改签其他日期或选择高铁等其他交通方式；
3. 通过阿里商旅平台操作退票或改签，优先选择航空公司提供的免费改签方案；
4. 及时通知主管和业务相关方，调整工作安排；
5. 保留所有相关凭证用于报销；
6. 如需紧急出行且无合适航班，可申请应急审批，选择其他交通方式。

### 紧急情况联系对象：
航班相关紧急情况需联系航空公司；同时需及时通知主管和业务相关方；若涉及差旅平台相关问题需要协助，可联系阿里商旅客服（电话：400-800-8888）。
